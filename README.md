NodeJS-WebixJet
===================

### How to run

- run ```npm install```
- run ```npm i formidable```
- run ```npm i sequelize```
- run ```npm i sqlite3```
- run ```npm i cors```
- run ```npm i body-parser```
- run ```npm i express```
- run ```npm run server```
- run ```npm start```
- open ```http://localhost:8080```
